HTML (Hypertext Markup Language) merupakan bahasa scripting untuk membangun website. HTML adalah bahasa dasar dalam pengembangan website. Setiap website pasti menggunakan HTML. Beberapa hal yang dapat dilakukan dengan HTML antara lain: modifikasi format teks, menampilkan daftar, membuat tautan, membuat tabel, memutar video dan audio, dan membuat form.
